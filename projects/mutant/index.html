mutant

Project overview

Mutant is an application that generates inspirational descriptions for the tabletop roleplaying game Mutant: Year Zero. In Mutant: Year Zero, players explore a post-apocalyptic wasteland called the “Zone,” sector by sector. The application can create Zone sectors with a single click, which is very handy during gameplay. It can also generate non-player characters (NPCs).
