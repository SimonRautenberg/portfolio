Project overview

This project is a text messenger that allows users to exchange messages between different browsers. Users need to log in because the messages are stored in a protected area. After logging in, each user receives a session ID to prevent unauthorized access. Messages are stored in the server’s backend in a database. When the messenger loads, the browser requests all existing messages from the server. Whenever a user sends a message, it is immediately transmitted to the server. The browser periodically checks the server for new messages and receives them as soon as they are available.
